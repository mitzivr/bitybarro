# Bit y Barro

## Descripción del Proyecto

Bit y Barro es una propuesta de negocio inclusivo desarrollada para la materia Diseñar para Compartir.

La iniciativa busca conectar artesanos mexicanos con mercados digitales mediante diseño, branding, fotografía profesional, empaques de valor agregado y comercio justo, permitiendo que los productores participen directamente en la cadena de valor y obtengan mayores beneficios económicos por su trabajo.

El nombre representa la unión de dos mundos:

- Bit: tecnología, diseño digital e innovación.
- Barro: tradición artesanal, patrimonio cultural y trabajo comunitario.

---

## Objetivo del Sitio Web

Crear una maqueta web accesible que presente el modelo de negocio y aplique los principios de accesibilidad digital de la metodología POUR:

- Perceptible (Perceivable)
- Operable (Operable)
- Comprensible (Understandable)
- Robusto (Robust)

## Cómo contribuir

1. Crear una cuenta de GitHub.
2. Solicitar acceso al repositorio.
3. Subir las imágenes correspondientes a la carpeta IMG.
4. Mantener una calidad visual consistente con la identidad del proyecto.

### Requisitos de las imágenes

- Formato JPG o PNG.
- Resolución mínima de 1200 px de ancho.
- Sin marcas de agua.
- Estilo profesional.
- Coherencia visual con artesanía mexicana contemporánea.
- Pueden ser fotografías reales o imágenes generadas con IA.
